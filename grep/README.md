# Build

Для сборки утилиты `grep` используйте:
```bash
$ make
```

# Tests

Для запуска тестов используйте (`python3` required):
```bash
make tests
```

# Usage

```bash
$ ./grep -help
Usage: grep: [OPTION]... PATTERN [FILE]...
       grep: [OPTION]... -e PATTERN [FILE]...
Author: Rustem Sirazetdinov
```
